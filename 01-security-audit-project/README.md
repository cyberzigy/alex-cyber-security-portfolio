# 🔐 Security Controls & Compliance Audit Assessment

## 📌 Project Overview

This project presents a security audit assessment of a simulated retail business environment. The objective was to evaluate existing security controls, identify operational and compliance risks, and recommend improvements to strengthen the organization’s security posture.

The assessment focused on administrative, technical, and physical safeguards aligned with common cybersecurity best practices.

---

## 🎯 Objectives

* Review current security controls
* Identify vulnerabilities and control gaps
* Evaluate regulatory compliance readiness
* Assess business risk exposure
* Recommend remediation actions

---

## 🔍 Audit Scope

The review covered:

* Employee devices (desktops, laptops, mobile devices)
* Internal network infrastructure
* Internet connectivity
* Data storage systems
* Payment systems
* Warehouse and storefront assets
* Legacy systems
* Physical security controls
* Backup and disaster recovery readiness

---

## ⚠️ Key Risks Identified

| Risk Area                              | Severity |
| -------------------------------------- | -------- |
| Excessive user access privileges       | High     |
| No disaster recovery plan              | High     |
| Weak password controls                 | Medium   |
| No centralized password management     | High     |
| No intrusion detection system          | High     |
| Unencrypted sensitive data             | High     |
| Inconsistent legacy system maintenance | Medium   |

---

## 🛡️ Existing Controls Reviewed

| Control                | Status              |
| ---------------------- | ------------------- |
| Firewall Protection    | Implemented         |
| Antivirus Protection   | Implemented         |
| CCTV Monitoring        | Implemented         |
| Physical Locks         | Implemented         |
| Fire Detection Systems | Implemented         |
| Password Policy        | Partially Effective |

---

## 📋 Compliance Considerations

Frameworks reviewed:

* PCI DSS
* GDPR
* SOC 1 / SOC 2

Findings indicated improvement was needed in:

* Data encryption
* Access control
* Password management
* Backup strategy
* Security monitoring

---

## ✅ Recommendations

1. Implement least privilege access controls
2. Deploy centralized password management
3. Encrypt sensitive customer data
4. Establish disaster recovery and backup procedures
5. Deploy IDS / SIEM monitoring solutions
6. Improve password complexity requirements
7. Create maintenance schedule for legacy systems
