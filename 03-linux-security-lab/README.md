# 🐧 Linux Security Lab

##  Project Overview

This project demonstrates practical Linux administration and security skills commonly used in cybersecurity environments. It focuses on system navigation, user management, file permissions, and basic security practices.

Linux is widely used in enterprise servers, cloud systems, and security operations, making foundational Linux knowledge essential for cybersecurity analysts.

---

##  Objectives

* Understand Linux file structure
* Manage users and groups
* Review and modify file permissions
* Monitor system activity
* Apply basic security practices

---

##  Skills Practiced

### System Navigation

* pwd
* ls
* cd
* mkdir
* rm
* cp
* mv

### User Management

* adduser
* passwd
* usermod
* groups

### File Permissions

* chmod
* chown
* ls -l

### Process Monitoring

* ps
* top
* kill

### Networking Basics

* ip a
* ping
* netstat
* ss

---

##  Security Concepts Applied

* Principle of least privilege
* Restricting unauthorized access
* Monitoring active processes
* Reviewing open network ports
* Securing file ownership

---

##  Skills Demonstrated

* Linux Command Line
* Access Control
* User Administration
* Process Monitoring
* Basic Network Troubleshooting
* Security Hardening Awareness

---

##  Relevance to Cybersecurity

Linux systems are common targets and hosts in enterprise environments. Understanding Linux helps analysts investigate incidents, review logs, detect suspicious activity, and secure servers effectively.

---

##  Conclusion

This lab demonstrates foundational Linux knowledge relevant to cybersecurity operations and prepares analysts for real-world environments where Linux systems play a critical role.
